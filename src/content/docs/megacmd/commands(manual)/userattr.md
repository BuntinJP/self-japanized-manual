ユーザー属性を一覧/更新します。

使用法: `userattr [-s attribute value|attribute|--list] [--user=user@email]`
<pre>
オプション:
 -s	attribute value	指定された属性に値を設定
 --user=user@email	問い合わせ対象のユーザーを指定
 --list	有効な属性を一覧表示
</pre>
